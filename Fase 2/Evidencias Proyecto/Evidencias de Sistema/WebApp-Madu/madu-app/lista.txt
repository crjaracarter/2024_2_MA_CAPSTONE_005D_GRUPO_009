Para listar tanto archivos como carpetas en Windows 11, aquí tienes varias opciones:

Usando PowerShell:

powershellCopy# Listar todo con detalles
Get-ChildItem

# Solo nombres
Get-ChildItem | Select-Object Name

# Con detalles específicos (nombre, tipo, tamaño)
Get-ChildItem | Select-Object Name, LastWriteTime, Length

# Guardar en un archivo
Get-ChildItem | Select-Object Name > lista.txt

Usando Command Prompt (CMD):

cmdCopy# Listar todo
dir

# Listar solo nombres
dir /B

# Guardar en archivo
dir /B > lista.txt
